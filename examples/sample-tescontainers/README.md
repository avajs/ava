# sample testcontainers with ava

Extending test boundaries by including the database

## Dependencies

- ava
- testcontainers postgresql module
- knex
- pg
- dotenv-flow

See [package.json][package.json] for details

## Running

Download dependencies:

```bash
npm i
```

Run the test suite:

```bash
npm run test
```
